# PolynomialInterpolation
Multi-point polynomial interpolation class in C++ for flexible data mapping.

* __Multi-Point Support__ : Allows setting up multiple coordinate pairs (value and adc) dynamically.
* __Segmented Calculation__ : Seamlessly finds the correct interval and computes interpolated values based on multiple data points.
* __Embedded-Friendly__ : Written in pure C++ with minimal overhead, suitable for microcontrollers and embedded platforms.
